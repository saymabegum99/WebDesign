Introduction

My idea for the Bangladeshiful website began while I was reading a book about the history of the Bengal. I was facinated by the depth of knowledge I had gained simply by reading but was also saddened by the fact that the bengali community itself knows very little about their own history. Especially the newer bengali generation in the UK. This is why I chose to make tihs website to educate the Bangladeshi community about Bangladeshi history and also chose to educate people about modern day Bangladesh by providing news articles. The structure was inspired by newspapers as they have columns to the side and a main article which is shown in the centre and the right of the page. The structure is very simple and easy to work with.

Inspiration

Three things mainly inspired me to make a website based on Bangladeshi news and history. The first inspiration began with reading a book on Bangladesh as mentioned previously. The book is called A Bangladesh history by William Van Schendel who explains the history behind Bangladesh dating as far back as the Mughal empire. It gives a great overview about the country and its vast history but also is not afraid to dive deeper make connections with geography and politics. It gave me the impression that Bangladesh is not just about the war of independance which many others believe it to be, but it is a country which lies within the largest delta in the world and thus has a large amount of history of different civilisations and tribes. I wanted others to learn about this as so many are unaware of their own ancestoral history.

Another inspiration for me was a documentary that I had seen. The documentary called The Bangladesh story which was made by the BBC, it gave me an in depth view of the stages which led up to the war and the after effects it had which have been still a problem with Bangladesh as of now. The latest article I had written on the website in fact explains this with the new general elections. I felt that it is very important history and context for people to understand the problems of the modern day and therefore I merged these two things together.

The final inspiration was my parents. They were only children during the war but They have taught me my mother tongue, took me to bengali classes after school and made sure that we visited our family in Bangladesh regularly growing up. This naturally drew me an interest in my own culture and therefore I thought it was only fitting for me to make a website because of this passion for history culture.

Accessibility

I tried to make the site accessible to all users and I have done this in three ways. I used the alt feature in image tags so that users who have a slow internet connection and the image fails to load for them can have a hint of what the image was to be. For example a picture of a field would be called ' A young girl running in a field with a Bangladeshi flag' Which I have written in one of my pages. The alt tag is also helpful for people who have a visual impairment, if they were to use text to speech the voice would read out the alt tag instead, so for this reason I tried to be descriptive as possible. I also tried to make the website accessbile by using the colours appropiately, I knew about colour theory and what colours go well together specifically for websites and text. This is useful to the visually impaired as they might find it hard to read text which had are similar in colour, or have issues with brightness. It is also helpful for those who are colour blind and have a difficulty to make out some colours. The social media tags I found had been small and could be hard to see for the visually impaired or the elderly, therefore I allowed the icons to enlarge by almost double the size so it wouldn't be an issue to read.

Usability

The website is very usable for users because I've added features like a navigation bar. This allows users to know and view the different pages they want. It also is helpful because it seperates the topics in the website and people know to go to which page to find the information they are looking for.

Another feature is the collapsible button, it allows users to navigate through the news page without being bombarded with a lot of text. Therefore the content in the boxes are kept nice and neat and so the user could have a easier time to click on the article title which they want to read without having to scroll down so much.

Another feature which makes this site very usable is the simplicity of the site. It is not very complicated and each section of the website is clearly distinct. I have colour coded each of the sections mainly green and grey so the users know that the left hand column and the section which is used to introduce the user to the page is mainly white and the bulk of the information is written with a background of pale green. This makes it easier for the user to navigate around the site.

Things I learnt

There are many things I learnt while creating my site. For example I learnt how to use the mapping tool. This is an excellent feature when you want to have interactive features in your website. I had checked an example in W3schools but the website did not mention how to go about knowing the positions for the image which you need while using the map feature. Therefore I had to use alternative methods, this is where I learnt that microsoft paint had a way of overcoming this problem and therefore I had used paint to work out the positions of the images.

I also learnt how to code the current date, this I learnt from a tutorial. I found it very helpful and suprising that Javascript had a feature date function which allows you to get the current date. I tried to find how to do this however I had trouble researching this using tradtional methods, looking at stackoverflow. I found them quite confusing and not very helpful as they tended to embed javascript in the html whereas I wanted the Javascript file to be external files. I then used a youtube video which was more like a tutorial and it was very easy to learn.

Learning how to make collapsible boxes using W3schools was very helpful. I've seen features like these in other websites and wanted to add it to my own. It allowed me to organise my website with articles as articles tend to be long and take up much of the space. Though I changed some of the code to allow the background to be transparent.

Things which worked well

The collapsible boxes which I put the articles I think were the best feature in the site. This is because the usefulness for the user and for me whilst using the site. As the site progresses with more articles, the articles can be clearly identified without the need to scroll down.I like how interactive this feature is too as users can select the articles they personally want.

Another aspect of the website which worked well is the fact that everything looks very uniform and similar. I like the layout of the website, it has a familiarity about it, like looking into a regular newspaper. The colour of the website and the structure gives it an overall appealing and calming look, I also had fun picking the gradient, colour scheme and logo for the website.

I only added pictures of the cities instead, but users could easily go online and do that. There would be also a real reason of why I added the map feature into the site too.
